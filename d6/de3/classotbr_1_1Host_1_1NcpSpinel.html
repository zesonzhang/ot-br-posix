Detailed Description

The class provides methods for controlling the Thread stack on the network co-processor (NCP).

This method sets the callback to receive the IPv6 address table from the NCP.

The callback will be invoked when receiving an IPv6 address table from the NCP. When the callback is invoked, the callback MUST copy the otIp6AddressInfo objects and maintain it if it's not used immediately (within the callback).

[in]aCallbackThe callback to handle the IP6 address table.
