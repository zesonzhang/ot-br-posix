Overrides MeshCoP service (i.e. _meshcop._udp) instance name, product name, vendor name and vendor OUI.

This method must be called before this BorderAgent is enabled by SetEnabled.

Parameters
[in]aServiceInstanceNameThe service instance name; suffix may be appended to this value to avoid name conflicts.
[in]aProductNameThe product name; must not exceed length of kMaxProductNameLength and an empty string will be ignored.
[in]aVendorNameThe vendor name; must not exceed length of kMaxVendorNameLength and an empty string will be ignored.
[in]aVendorOuiThe vendor OUI; must have length of 3 bytes or be empty and ignored.
[in]aNonStandardTxtEntriesNon-standard (vendor-specific) TXT entries whose key MUST start with "v"
Returns
OTBR_ERROR_INVALID_ARGS If aVendorName, aProductName or aVendorOui exceeds the allowed ranges or invalid keys are found in aNonStandardTxtEntries
OTBR_ERROR_NONE If successfully set the meshcop service values.
